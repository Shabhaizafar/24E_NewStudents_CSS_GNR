Image Transform Variants (Instructions)

Translate: On hover, apply transform: translate(30px, -20px);

Design instruction: Set image size to 200Ã—200px, border radius to 16px, and apply a soft white box-shadow. Center the image using text-align center. Use a dark background and light text for contrast.

Scale: On hover, apply transform: scale(1.3, 1.1);

Design instruction: Same as above. This effect makes the image appear larger when hovered.

Rotate: On hover, apply transform: rotate(25deg);

Design instruction: Rotates the image slightly to give a dynamic effect. Use smooth transition for better appearance.

Skew: On hover, apply transform: skew(15deg, 10deg);

Design instruction: This slants the image for a stylistic appearance. Apply consistent border radius and box shadow as above.
